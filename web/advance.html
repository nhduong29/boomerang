#1 Advance mode

*** Please note that we run those commands below on the Unix platform. If you are using windows, please open the git bash to run it

#1.1 Options

This is the standard command line to run the application

node index.js --port=9004 log/debug.log -s sandbox-lib -x 9003
  • index.js is the root file of the application
  • --port is the listening port of UI streaming request, default is 9004
  • --s is the location of sandbox lib, default is sandbox-lib
  • --x is the port of the API webservice, default is 9003

#1.2 API Documentation

All API Definition include inside the file sandbox-tail/services/main.js

You can import other javascript files by this line:

require("./path/fileName.js");

More detail:

  • https://getsandbox.com/docs/require-files

About the API definition, please follow the syntax of Sandbox:

  • https://getsandbox.com/docs/definition-api

#1.3 Standalone application

Install Forever, a lib to make the node js app run permanently:

sudo npm install -g forever

Start the application by this command:

forever start index.js --port 9004 log/debug.log -s sandbox-lib -x 9003

The options of Forever you can see more details here: https://www.npmjs.com/package/forever
