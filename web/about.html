#4 Usage

*** Please note that we run those commands below on the Unix platform. If you are using windows, please open the git bash to run it

Go to root folder of Boomerang project:

$ cd boomerang_dcrab/sandbox-tail

Install all nodejs dependencies:

$ sudo npm install

Start the application in one line of code:

node index.js --port=9004 log/debug.log -s sandbox-lib -x 9003

Web interface runs on http://localhost:9004

The API webservice run on http://localhost:9003

Now try to use Postman or create a GET request to this URL: http://localhost:9003/hello-world

Then open the web browser to see the amazing world : http://localhost:9004
