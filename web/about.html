Options

  • -h, --help output usage information
  • -p, --port listening port, default 9004
  • -s or --sandbox is the place of sandbox service, default sandbox-lib
  • -b or --base is the place of sandbox API Documentation, default services
  • -x or --sandbox-port is the port of sandbox service, default 9003

Usage

  • npm install
  • node index.js --port=9004 log/debug.log -s sandbox-lib -x 9003
  • Sandbox URL http://localhost:9003
  • Web interface runs on http://localhost:9004
